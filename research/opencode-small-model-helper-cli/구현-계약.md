# OpenCode helper CLI 구현 계약

## 목적

Qwen3.6급 small model이 OpenCode 안에서 broad `bash`에 기대지 않고, 좁고 검증 가능한 helper tool을 먼저 사용하게 만드는 계약이다. 이 문서는 구현 시 tool 이름, 입력 schema, 출력 형태, permission 기본값을 고정하기 위한 기준이다.

## Tool 목록

| Tool | 내부 CLI | 기본 권한 | 목적 |
| --- | --- | --- | --- |
| `helper_find_files` | `fd` | `allow` | 파일/디렉터리 후보를 bounded list로 반환 |
| `helper_search_text` | `rg --json` | `allow` | text match를 normalized JSON으로 반환 |
| `helper_json_query` | `jq` | `allow` | JSON에서 필요한 field만 projection |
| `helper_yaml_query` | Mike Farah `yq` v4 | `allow` | YAML/JSON/TOML/XML 등 구조화 파일 query |
| `helper_markdown_query` | `yshavit/mdq` | `allow` 또는 optional | Markdown heading/task/link/table/front matter query |
| `helper_ast_search` | `ast-grep` | `allow` | AST 구조 검색 |
| `helper_ast_rewrite_preview` | `ast-grep` | `ask` | rewrite preview만 생성, 실제 apply 금지 |

## 공통 입력 규칙

- 모든 path는 workspace root-relative path만 허용한다.
- `..`, absolute path, symlink root escape는 거부한다.
- `maxMatches`, `maxFiles`, `maxBytes`를 필수 또는 기본값으로 둔다.
- wrapper는 shell string이 아니라 argv array로 CLI를 실행한다.
- stdout은 raw text 그대로 반환하지 말고 typed JSON으로 normalize한다.
- error는 `status`, `exitCode`, `stderrSummary`, `hint`를 포함해 반환한다.

## 권장 schema 개요

```ts
helper_find_files({
  root?: string,
  pattern?: string,
  extensions?: string[],
  type?: "file" | "dir" | "any",
  includeHidden?: boolean,
  maxFiles?: number
})
```

```ts
helper_search_text({
  pattern: string,
  paths?: string[],
  globs?: string[],
  caseSensitive?: boolean,
  maxMatches?: number
})
```

```ts
helper_json_query({
  path?: string,
  input?: string,
  expression: string,
  output?: "compact" | "pretty" | "raw",
  maxBytes?: number
})
```

```ts
helper_yaml_query({
  path?: string,
  input?: string,
  expression: string,
  output?: "json" | "yaml" | "raw",
  implementation: "mikefarah-v4"
})
```

```ts
helper_ast_search({
  language: string,
  pattern: string,
  paths: string[],
  maxMatches?: number
})
```

```ts
helper_ast_rewrite_preview({
  language: string,
  pattern: string,
  rewrite: string,
  paths: string[]
})
```

## Bootstrap 기준

필수:

- `rg`
- `fd`
- `jq`
- Mike Farah `yq` v4

권장:

- `ast-grep`
- `yshavit/mdq`

선택:

- `ast-grep-mcp`

`ast-grep`와 `mdq`를 helper tool로 노출하려면 installer 또는 doctor에서 설치 여부를 반드시 검사한다.

## Small model 사용 규칙

Small model prompt에는 아래 규칙을 넣는다.

```text
1. 먼저 typed helper tool을 사용한다.
2. helper가 없을 때만 bash를 사용한다.
3. 결과는 evidence ref, command, output summary, uncertainty를 포함한다.
4. rewrite는 preview만 수행하고 apply는 별도 승인 뒤에만 한다.
5. JSON/YAML/Markdown은 text slicing보다 구조화 query를 우선한다.
```
